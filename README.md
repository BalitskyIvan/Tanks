# Run client

To run client you shoud be in root directory (TanksClient/) and run:

    mvn
    
# Run server

You need to run your postgresql database and write your settings in db.properties,
Also you shoud import database schema from schema.sql

To run client you shoud be in root directory (TanksClient/) and run:

    mvn
